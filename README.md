# Daily Report Program

This program performs a daily task at a specified time, similar to a cron job. It accesses your server through Chrome Developer, takes screenshots, and sends them to multiple numbers via WhatsApp.

## Installation

1. **Create and Activate `.env` File**
   - Create a `.env` file and activate it.

2. **Update Twilio Credentials**
   - Update the Twilio credentials in the `.env` file.

3. **Install Requirements and Run the Program**
   - Install the necessary requirements and run the program.
